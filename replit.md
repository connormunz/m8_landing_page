# m8_landing_page

A simple static personal landing page (module 8, BAIS3500).

## Project Structure

- `index.html` — The main (and only) HTML page

## Tech Stack

- Plain HTML (no build system, no frameworks)

## Development

The site is served using Python's built-in HTTP server on port 5000.

**Workflow:** `Start application` — `python3 -m http.server 5000 --bind 0.0.0.0`

## Deployment

Configured as a static site deployment with the project root as the public directory.
