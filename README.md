## 1. About

This is a personal professional landing page built for Connor Munz, a Business Analytics and Information Systems student at the University of Iowa — Tippie College of Business, graduating December 2026. The page exists to give recruiters, hiring managers, and Connor's personal network a curated, living snapshot of who he is professionally — going beyond what a static résumé can convey. It highlights his technical skills, real project work, and creative approach to data-driven problem solving. The site is deployed as an Azure Static Web App with a custom domain, and version-controlled through GitHub.

## 4. Features

- **Hero section** with name, professional tagline, and a call-to-action button linking to the contact section
- **About / Bio section** with academic background, GPA, Dean's List recognition, and a personal note on creative problem-solving
- **Skills section** with color-coded pill badges grouped by category (Programming, Technical Skills, Techniques)
- **Projects section** featuring the Spotify SQL Data Analysis project with tools and outcomes described
- **Experience section** highlighting roles at RSM US LLP, Accertify (2025 Intern Fraud Champion), and OV Solutions
- **Contact section** with linked badges for LinkedIn, GitHub, and email — all opening in a new tab
- **Sticky navigation bar** with smooth scroll anchor links to each section
- **Fully responsive layout** — single column on mobile, two-column project grid on desktop
- **WCAG 2.2 Level AA accessibility** — descriptive alt text, logical heading hierarchy, keyboard navigable
- **Navy and teal color palette** with Inter typography for a clean, professional aesthetic
